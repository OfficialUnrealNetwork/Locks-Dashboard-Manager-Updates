Administration

Settings

Control appearance, reports, local storage, data export, and safety options for the dashboard.

Appearance

Theme and display
Dark Mode Switch the whole dashboard to a darker theme. This setting is saved on this device.
Time Format Choose whether clocks and report times display in 12-hour or 24-hour format.
Compact Tables Use tighter table rows for screens with less space.
Reduce Animations Turn down hover movement and visual motion across the app.

Entry Defaults

New record behavior
Default Canal Used as the default canal value for new entries when the form supports it.
Default Direction Preferred direction for lockage forms.
Auto-fill Current Time Use the current time when opening a new entry form.
Confirm Before Saving Show a confirmation step before saving new records.

Reports

Print and export preferences
Default Report Choose which report type opens first on the All Records page.
Print Theme Printed reports should stay light even when the dashboard is dark.
Report Density Controls how tight report previews and printed tables appear.
Show All Traffic Types Daily, monthly, and range reports show TB, RB, Government, Commercial, LR, and LT totals.

Data Tools

Backup and reset

This dashboard saves records in this browser using localStorage. Export backups before resetting data or moving to another computer.

Export Backup Download the current local database as a JSON backup file.
Import Backup Load a previously exported JSON backup into this browser.
Export Today Download only today’s records as a JSON file.
Danger Zone Resetting demo data removes locally saved records from this browser. Export a backup first.
